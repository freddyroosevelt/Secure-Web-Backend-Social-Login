# Secure-Web-Backend-Social-Login

## Secure backend setup for a user login to a web app.

### Purpose of web-app

* Secure backend for a web app
* User login with encrypted data
* Each user has its own protected user token
* User signup/login with social accounts(Facebook/Google+)
* User signup/login with email/password

#### Backend Dependices used

* Node.js
* Express
* MongoDB
* Passport
* Bcrypt
* Rand-Token
* Passport-Http-Bearer
